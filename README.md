# secreate.io

Тестовое задание от https://secreate.io/ 


# Ветки разработки

  client-master - Фронт
  
  server-master - Сервер
  
  master - docker-compose ( запуск всей системы )

# Запуск


docker-compose build - создание

docker-compose up - Запуск серверов ( mongo, back-end, front-end )

docker-compose down
